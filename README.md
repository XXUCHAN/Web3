# 🖥️ 개발 배경 🖥️
**마이크로 자원봉사**에 관한 접근성 향상을 통한 봉사 인식 변화를 도모하고, <br/>
봉사의 생활화를 통한 선한 영향교내 장애인 학우를 위한 **봉사 활동 장려 웹 서비스**의 개발

NFT를 통한 리워드 차등 지급으로 봉사에 대한 참여도를 향상시켜 종합적으로는 교내 장애인 학우를 위한 봉사 활동 장려 웹 개발을 하게 됐습니다.<br/><br/>
